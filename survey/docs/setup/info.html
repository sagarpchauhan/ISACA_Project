Software Features

Easily create website surveys
Create and attach colour schemes to surveys
Built in colour schemes to get you started
Easily add questions to surveys
SMTP mail send option (Optional)
Copy surveys (Optional)
Export contacts list to CSV format (Optional)
Keyword analysis feature with option to omit popular words
View keyword analysis for all questions in survey, or single questions or group by date
Survey display option. Load a survey on a single page or multiple pages
Set a title for your survey (Optional)
Input data filtered from survey answers to prevent harmful code execution
Request particpants contact details after survey completion (Optional)
Enable participants to view survey results (Optional)
Set expiry date for surveys (Optional)
Display message after survey completion or direct to specific URL
Enable participants to complete survey multiple times (Optional)
Receive e-mail notification (can be sent to more than one address) if survey completed (Optional)
Specify order in which questions appear
Provide optional help text for survey questions
Questions can have several types of response depending on the type of question
Optional captcha for single page surveys. Captcha can be survey specific
Make question fields compulsory
User friendly and cosmetic admin interface
Template driven system to enable you to add surveys to your existing website
MySQL backend for powerful, robust usage
Auto install file. Easy to set up.
